# intelligent-calendar-for-BUAAer

## 智能日程助手架构（北航版）

### 系统核心组成

本系统主要由网页前端和Python后端构成，旨在为北航学生提供一个高效、智能的日程管理工具。

**1. 前端部分（Web界面）**
*   **主视图区**：展示七日视图日历。
*   **任务管理侧边栏**：清晰列出所有的作业DDL和待办事项，支持直接勾选完成。
*   **智能输入面板**：支持文本输入、图片上传，并自动检测剪切板内容。
*   **设置面板**：提供北航账号绑定、提醒偏好设置（为不同类型的任务配置不同的提前提醒时间）、主题外观等个性化选项。

**2. 后端部分（Python服务）**
*   **用户与数据管理**：处理用户信息，并通过北航官方API获取并同步课程、考试等固定日程。
*   **智能解析中心**：处理用户输入的文本和图片。文本解析直接进行，图片中的课程表或任务信息可考虑使用成熟的OCR服务进行识别。所有解析出的信息均交给大语言模型进行意图识别和任务提取。
*   **日程管理引擎**：负责创建日程、检测简单的日程冲突（如时间重叠），并提供冲突提示，交由大语言模型自动处理冲突或由用户手动调整。
*   **提醒与通知系统**：根据任务类型和用户设置，在预定时间通过浏览器通知等方式发送提醒。

### 核心功能流程

**1. 课程表自动获取**
*   用户绑定北航学号后，系统通过北航API获取课程和考试数据，并自动填入日程。这避免了手动录入的繁琐，确保了信息的准确。

**2. 智能输入与任务创建**
*   系统支持多种输入方式，并能自动检测剪切板中的文本内容（例如，复制的作业通知）。
*   这些文本或图片经OCR识别后的文本，会发送给大语言模型进行解析。模型需要识别出任务类型（如作业、讲座）、截止日期、预估耗时等关键信息。

**3. 任务分类与智能安排**
*   **固定时间任务**：如课程、考试，直接在日历的指定时间创建，并根据用户设置（如课前15分钟）添加提醒。
*   **弹性DDL任务**：如作业，系统会做两件事：
    *   在DDL日期创建一个最终截止提醒。
    *   根据课程难度、预估耗时以及用户设置的最佳学习时间段，在DDL之前的空闲时段自动安排一个或多个“推荐完成时段”。用户可自由调整或取消此推荐时段。

**4. 个性化提醒设置**
*   用户可以为不同类型的任务设置不同间隔的提前提醒。例如：
    *   **课程**：提前30分钟提醒。
    *   **考试**：支持多级提醒（考前一周、前一天、当天早上）。
    *   **作业DDL**：提前24小时再次提醒。
    *   **博雅活动**：提前2小时提醒。
*   这些设置让提醒更具人性化，符合不同任务的实际需求。

**5. 任务进度跟踪**
*   在侧边栏的DDL清单中，用户可以勾选已完成的任务。
*   系统可视化和统计已完成和未完成的任务，例如通过一个进度条。对于未在计划内完成的任务，系统会提示用户是否需要重新安排时间。
